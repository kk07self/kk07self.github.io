Audio Player

Demo介绍

  • 业务流程：音频选择——》音频播放（伴随动效）

    • 音频选择：ViewController，ChooseAudioViewController

      这里提供了四个入口，iCloud是需求内的，其他的三个是在研究技术点的过程中接触到的，一个循序渐进的过程吧。

      • iCloud—–系统UI——UIUIDocumentViewController
      • 音乐选择—–系统UI—–MPMediaPickerViewController
      • 音乐选择—–非系统UI（ChooseAudioViewController）
      • 项目中的音频demo：这个是最开始调试播放器的时候直接添加到项目中的

  • 音频播放：AudioPlayerViewController

    音频播放的主要控制器，音频的播放器用的是AVAudioPlayer，目前现实的是播放、暂停功能

  • 音频播放动效：VIWaveformView

    这个是在github上找的一个库，但是动效和我们的demo不一样，这边就将其内部的动效绘制进行了修改，以达到demo的效果。VIWaveformView地址

    当然还有很多其他比较好的库，比如：EZAudio

​ 其他个别辅助，是以前开写得小工具就直接拿过来用了

测试注意

因为有用到iCloud这个功能，所以在账号Team那里最好用组织或公司开发账号，我一开始用的个人的非付费用户是不行的
